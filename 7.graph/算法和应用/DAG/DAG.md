# 有向无环图

- 一个无环的有向图被称为有向无环图(Directed Acyline Graph),

- DAG 本质上是用来表达“先后依赖”或“部分有序”关系的数学模型。
它把“谁先谁后”抽象成节点和边，只要图中无环，就说明这套依赖是自洽的（不会 A 依赖 B，B 又间接依赖 A）。

例如:

```
graph TD
    A['*'] --> B['+']
    A --> C['*']
    B --> D['*']
    B --> E['*']
    D --> F['+']
    D --> G['*']
    F --> H['a']
    F --> I['b']
    G --> J['b']
    G --> K['+']
    K --> L['c']
    K --> M['d']
    E --> N['+']
    E --> O['e']
    N --> P['c']
    N --> Q['d']
    C --> R['+']
    C --> S['e']
    R --> T['c']
    R --> U['d']
```

 可以通过DAG简化为:

```
graph TD
    A['*'] --> B['+']
    A --> E['*']
    B --> D['*']
    B --> E['*']
    D --> F['+']
    D --> G['*']
    F --> H['a']
    F --> J['b']
    G --> J['b']
    G --> N['+']
    E --> N['+']
    E --> O['e']
    N --> P['c']
    N --> Q['d']
```
