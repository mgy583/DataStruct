# 树

## 树的基本概念

### 定义

树是一种**非线性**数据结构，由( n >= 0 )个节点组成的有限集合:

- 若 n = 0 则为空树
- 若 n > 0 则:
  - 有且只有一个根节点(Root)
  - 其余节点 m (m > 0) 个互不相交的有限集合，每个集合本身有是一颗树，成为子树

### 树的属性

| 属性 | 定义 |
| 节点(Node) | 树的基本单元，包含所带数据和指向其他节点的指针 |
| 根节点(Root) | 树的最顶层节点 |
| 父节点(Parent) | 当前节点的上层节点 |
| 字节点(Child) | 当前节点的下层节点 |
| 兄弟节点(Siling) | 和当前节点具有相同父节点的节点 |
| 叶节点(Leaf) | 没有子节点的节点(度为0) |
| 内部节点 | 至少有一个子节点的节点 |
| 度(Degree) | 节点拥有的子树数量(子节点数) |
| 树的度 | 树中所有节点度的最大值 |
| 层次(Level) | 树的第一层向下递增 |
| 深度 | 树中节点的最大层次 |
| 路径 | 从根节点到某节点所经过的节点序列 |
| 祖先\子孙 | 从根到某节点的路径上的所有节点是该节点的祖先;反之为其子孙 |
| 森林 | 多颗互补相交的树的集合 |
| 有序树\无序树 | 树中节点的各子树依次从左到右有序\无序,可以交换\不可交换

## 常考性质

### 考点1 - 节点数 = 总度数 + 1
  
节点的度： 节点有几个分支
  
### 考点2 - 度为m的树和m叉树的区别

树的度 - 各节点的度的最大值

- 度为 m 的树:
  - 任意节点的度 < m (最多 m 个孩子)
  - 至少有一个节点的度为 m (有 m 个孩子)
  - 一定是非空树, 至少有 m + 1 个节点

m 叉树 - 每个节点最多只能有 m 个孩子的树

- m 叉树:
  - 任意节点的度 <= m (最多 m 个孩子)
  - 允许所有节点的度都 < m
  - 可以是空树

### 考点3 - 树的 i 层**最多**节点数
  
- 度为 m 的树的第 i 层 **至多** 有 m^(i-1) 个节点(i>=1)
- m 叉树 的第 i 层 **至多** 有 m^(i-1) 个节点(i>=1)

### 考点4 - 高度 h 叉树 m 的树**至多**节点数

- 高度 h 叉树 m 的树至多有 m^h-1/m-1 个节点

### 考点5 - 高度为 h 的 m 叉树**至少节点**

- 高度为 h 的 m 叉树至少节点有 h 个节点
- 高度为 h、度为m的树至少有 h+m-1 个节点

### 考点6 - 具有 n 个节点的 m 叉树的**最小高度**

- 具有 n 个节点的 m 叉树的最小高度 **log_m(n(m-1)+1)**
